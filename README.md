# Voss Ekobi Personal Portfolio Website

Мультидисциплінарний портфоліо сайт, що поєднує слово, звук та візуальні образи.

## 🌟 Особливості

- **Темна/Світла тема** - Адаптивний дизайн з переключенням тем
- **Багатомовність** - Підтримка української та англійської мов
- **Адаптивний дизайн** - Оптимізовано для всіх пристроїв
- **Гладкі анімації** - Динамічні переходи та ефекти
- **Модальні вікна** - Розширене читання історій з контролем розміру шрифту
- **Музичний плеєр** - Інтегрований мініатюрний плеєр для композицій
- **Галерея з фільтрами** - Сортування контенту по категоріям
- **Сучасний стек** - HTML5, CSS3, Vanilla JavaScript, Lucide Icons

## 📁 Структура

```
vossekobi.github.io/
├── index.html      # Основна сторінка
├── styles.css      # Стилі
├── script.js       # JavaScript функціональність
└── README.md       # Цей файл
```

1. Клонуйте репозиторій
2. Відкрийте `index.html` у браузері
3. Насолоджуйтесь портфоліо!
